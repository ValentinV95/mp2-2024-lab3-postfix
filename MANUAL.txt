Данная программа может переводить математические выражения в постфиксную фому и вычислять их значения при заданных параметрах.
На ввод накладываются следующие ограничения:
0. Выражения, не имеющие математического смысла не обрабатываются;
   Число пробелов между операндами (здесь и далее операндом считается константа, переменная, самостоятельное выражение),
   операциями, функциями, скобками может быть любым;
1. Числовые константы
	- целая часть должна быть написана явно (не допускается, например, такой вид: .5);
	- дробная часть отделяется от целой точкой и только ей, если её нет, то считается что нет и дробной части;
	- после точки обязательно должна следовать дробная часть;
	- доступна научная форма записи с показателем (пример: 25e+3 (=25'000), 1.5e-1 (=0.15));
	- знак показателя должен быть указан явно (не допускается, например, такой вид: 5e1);
	- показатель должен быть целым;
	- на длину числа и количество незначащих нулей ограничений нет;
	- ОДЗ числовых констант ограничено множеством значений типа double;
2. Переменные
	- переменная должна начинаться либо с латинской буквы, либо со знака нижнего подчеркинвания '_', если в переменной есть цифры, то
они должны непрерывно располагаться в конце названия переменной (играют роль индекса). Переменные вида ab1c2 не допускаются;
	- имена переменных чувствительны к регистру;
	- длина переменной не может превышать 16 символов;
	- для математических констант зарезервированы имена: pi и e (Pi и E свободны);
	- имена переменных не должны совпадать с названиями функций (см. далее);
3. Математические операции
	- доступны следующие операции: +, -, *, /, ^ (возведение в степень), круглые скобки;
	- операция ^, как и все бинарные, левоассоциативная, т.е. выражение a^b^c = (a^b)^c = a^(b*c);
	- доступна операция унарного минуса, число унарных минусов перед одним операндом или скобкой неограничено,
между минусами может быть сколько угодно пробелов и не может быть других операций;
	- для операций установлены следующие приоритеты: +,- < *,/ < унарный - < ^;
	- любая открывающая скобка должна иметь свою закрывающую скобку;
	- скобки не могут быть пустыми;
	- не допускается отсутсвие операции между операндом и операндом/скобкой;
4. Элементарные функции
	- на выбор предоставляются следующие функции:  abs, log, sin, asin, sinh, asinh,
cos, acos, cosh, acosh, tan, atan, tanh, atanh, cot, acot, coth, acoth;
	- функция exp(x) доступна через e^x;
	- аргумент функции обязательно должен содержаться в скобках;

Правила работы с программой:
	- при первом открытии программа предложит вам прочесть данное руководство;																									вы молодец, если сразу его прочитали
	- программа предложит ввести выражение;
	- при неверно введённом выражении будет выдано сообщение об ошибке с указанием на позицию, с которой начинается проблема ввода.
Позиция считается с 0 - с первого введённого символа. Первый пробел и последний не считаются. Остальные пробелы принимаются за один символ;
После можете ввести выражение повторно с поправленными ошибками;
	- при успешно введённом выражении будут выведены само выражение и его постфиксная форма, далее необходимо ввести значения
переменных. Для каждого имени вводится одно значение, в порядке выведения их имён, хотя пользователь может ввести одной стройчкой все
переменные сразу. В таком случае порядок введения соответствует порядку появления переменных в строке;
	- пользователь сам контролирует корректность ввода значений переменных;
	- после введения значения всех переменных будет выведен результат выражения. Если в результате вычисления выражение приобрело
очень большое значение, будет выведено infinity с указанием знака. Если во время вычислений произошёл выход за границы вещественных чисел
или неопределённость вида infinity/infinity, 0*infinity, и им подобные, будет выведено undefined;
	- после выведение результата программа предложит вычислить выражение для других значений переменных. Для подтверждения нажмите y.
Иначе нажмите n;
	- при отказе программа предложит ввести другое выражение. Для подтверждения нажмите y. Иначе нажмите n. Последнее приведёт к
выходу из программы;
	- программа автоматически вычисляет выражения состоящие только из числовых и математических констант;
	- программа может принять на вход и один операнд-константу или переменнуюю. В результате вычисления будет выведено его значение.
